# Angular Basics

This repository was created to document my first studies and experiments with Angular 19.

## 📚 Topics Covered

- Standalone Components
- Interpolation
- Property Binding
- Event Binding
- Form Handling
- List Rendering
- SCSS Styling

## 🧩 Components Developed

- HelloWorld
- Calculator
- Shopping List
- My Page

## 🚀 Running the Project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
ng serve
```

The application will be available at:

```text
http://localhost:4200
```

- Angular CLI
